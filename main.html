Python для Minecraft

Написание скриптов, автоматизация и творчество в кубическом мире

Minecraft — это не только строительство и выживание, но и мощная платформа для обучения программированию. Используя Python, вы можете создавать скрипты, которые строят огромные конструкции, управляют игровыми сущностями, генерируют миры и даже создают мини-игры. В этой статье разберём основы написания скриптов на Python для Minecraft, познакомимся с библиотеками и напишем первый рабочий код.

Почему Python?

Python — идеальный выбор для автоматизации в Minecraft благодаря простоте, читаемости и богатой экосистеме. С его помощью даже новичок может за несколько минут написать скрипт, который построит пирамиду или телепортирует игрока. А опытные разработчики используют Python для создания сложных модов, ботов и целых серверных дополнений.

Необходимые инструменты

  • Minecraft Java Edition (версия 1.12+ или 1.16+ для большинства библиотек)
  • Raspberry Juice или Spigot/Paper с плагином RaspberryJuice — мост между Python и игрой
  • Библиотека mcpi (MCPI) — основной инструмент для управления миром
  • Python 3.8+ и любой редактор кода (VS Code, PyCharm или даже Блокнот)

RaspberryJuice — это плагин для сервера Bukkit/Spigot, который эмулирует API Minecraft: Pi Edition. После его установки Python-скрипты могут подключаться к серверу через сокеты и выполнять команды: ставить блоки, перемещать игрока, считывать позицию.

Совет — Если вы хотите попробовать без установки сервера, используйте Minecraft Pi edition на Raspberry Pi или эмуляцию через mcpipy. Но полноценный сервер даёт больше возможностей.

Установка и настройка

Пошаговая инструкция для быстрого старта:

  1. Скачайте и установите Java Edition Minecraft.
  2. Загрузите сервер Spigot 1.16.5 (или Paper) и запустите его один раз, чтобы создались папки.
  3. Скачайте RaspberryJuice — jar-файл плагина с GitHub (последняя версия). Поместите его в папку plugins/ сервера.
  4. Перезапустите сервер. Плагин активирует порт 4711 для подключения Python.
  5. Установите библиотеку mcpi в Python: pip install mcpi.
  6. Подключитесь к своему серверу (локальный или по IP) из Minecraft и запустите первый скрипт.

Готово! Теперь вы можете управлять миром из Python.

Ваш первый скрипт: приветствие и алмазный блок

Давайте создадим скрипт, который подключается к игре, отправляет сообщение игроку и размещает блок алмаза прямо перед ним.

from mcpi.minecraft import Minecraft
import time

# Подключаемся к Minecraft серверу (по умолчанию localhost:4711)
mc = Minecraft.create()

# Получаем координаты игрока
player_pos = mc.player.getTilePos()
x, y, z = player_pos.x, player_pos.y, player_pos.z

# Отправляем сообщение в чат
mc.postToChat("Привет, Minecraft! Это Python скрипт")

# Ставим алмазный блок прямо перед игроком (смещение по оси X)
mc.setBlock(x + 1, y, z, 57)   # 57 — ID алмазного блока

# Строим небольшую колонну из золота над алмазом
for i in range(1, 4):
    mc.setBlock(x + 1, y + i, z, 41)   # 41 — золотой блок
    time.sleep(0.3)

mc.postToChat("Готово! Алмаз и золотая колонна созданы.")

Сохраните этот код в файл first_minecraft.py. Зайдите в мир Minecraft (на сервер с плагином), выполните скрипт — и вы увидите, как в игре появляются блоки, а в чате появляются сообщения.

Основы MCPI API

Библиотека mcpi предоставляет интуитивные методы для взаимодействия с миром. Рассмотрим ключевые возможности:

Работа с игроком

pos = mc.player.getPos()
mc.player.setPos(x, y, z)
mc.player.setTilePos(x, y, z)

Блоки

mc.setBlock(x, y, z, block_id)
mc.setBlocks(x1,y1,z1,x2,y2,z2, block_id)
block = mc.getBlock(x, y, z)

Чат и события

mc.postToChat("текст")
events = mc.events.pollBlockHits()

Полный список ID блоков можно найти в вики Minecraft (например, 1 — камень, 3 — земля, 5 — древесина, 35 — шерсть). Для цветных блоков используйте дополнительные данные (metadata).

Практические примеры скриптов

Рассмотрим три полезных скрипта: автоматическое строительство пирамиды, телепортация по координатам и создание летающего острова.

1. Строитель пирамиды из песчаника

from mcpi.minecraft import Minecraft
mc = Minecraft.create()

def build_pyramid(base_x, base_z, height, block_id):
    # Получаем уровень земли из текущей позиции игрока
    player = mc.player.getTilePos()
    base_y = player.y
    
    for level in range(height):
        size = height - level
        offset = level
        for i in range(size):
            for j in range(size):
                mc.setBlock(base_x + i + offset, base_y + level, base_z + j + offset, block_id)
    mc.postToChat("Пирамида готова!")

# Построим пирамиду высотой 5 из песчаника (24)
build_pyramid(0, 0, 5, 24)

2. Телепорт по имени игрока

from mcpi.minecraft import Minecraft
mc = Minecraft.create()

def teleport_to_player(target_name):
    players = mc.getPlayerEntityIds()
    for pid in players:
        if mc.entity.getName(pid).lower() == target_name.lower():
            pos = mc.entity.getTilePos(pid)
            mc.player.setTilePos(pos.x, pos.y + 2, pos.z)
            mc.postToChat(f"Телепортировались к {target_name}")
            return
    mc.postToChat("Игрок не найден")

teleport_to_player("Steve")

3. Летающий остров (сфера из камня)

from mcpi.minecraft import Minecraft
import math
mc = Minecraft.create()

def sphere(center_x, center_y, center_z, radius, block_id):
    for x in range(-radius, radius+1):
        for y in range(-radius, radius+1):
            for z in range(-radius, radius+1):
                if x*x + y*y + z*z <= radius*radius:
                    mc.setBlock(center_x + x, center_y + y, center_z + z, block_id)

# Создаём каменную сферу радиусом 7 на высоте 70
sphere(0, 70, 0, 7, 1)
mc.postToChat("Остров-сфера создан!")

Все эти скрипты можно комбинировать, добавлять циклы и условия — возможности ограничены лишь фантазией.

Продвинутые сценарии: события, анимация и боты

MCPI позволяет обрабатывать события — например, когда игрок разрушает блок или нажимает на кнопку. Вы можете написать реактивный скрипт, который реагирует на действия в реальном времени.

from mcpi.minecraft import Minecraft
import time

mc = Minecraft.create()
mc.postToChat("Активирован детектор блоков. Ломай блоки!")

while True:
    events = mc.events.pollBlockHits()
    if events:
        for hit in events:
            x, y, z = hit.pos.x, hit.pos.y, hit.pos.z
            mc.postToChat(f"Сломан блок на координатах {x}, {y}, {z}")
            # Восстанавливаем блок через секунду
            time.sleep(0.5)
            mc.setBlock(x, y, z, 1)  # камень
    time.sleep(0.2)

Также вы можете создавать анимированные конструкции, световые шоу, или написать простого бота, который следует за игроком и строит дорожку из розовой шерсти.

Полезные советы и частые ошибки

  • Соединение отклонено: убедитесь, что сервер запущен, плагин RaspberryJuice активен и порт 4711 не блокируется фаерволом.
  • Блоки не ставятся: проверьте, что вы находитесь в области загрузки чанков (не слишком далеко от спавна). Используйте mc.player.getTilePos() и стройте рядом.
  • ID блоков изменились: в новых версиях Minecraft некоторые ID могут отличаться. Используйте числовые идентификаторы из документации плагина или константы.
  • Производительность: не ставьте тысячи блоков в одном цикле без time.sleep — это может вызвать лаги. Разбивайте операции на части.
  • Многопоточность: библиотека mcpi не потокобезопасна, лучше использовать асинхронность или последовательные вызовы.
